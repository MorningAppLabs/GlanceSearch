# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-14

### Added
- System-wide shortcut (`Ctrl+Shift+G`) for instant screen capture
- Optical Character Recognition (OCR) using Windows 10/11 built-in engine
- Smart floating Action Panel 
- Content detection for URLs, Emails, Phone Numbers, and Barcodes/QR Codes
- Inline Text Translation with support for Free Tier (MyMemory) and Premium (DeepL, Google, Azure)
- Search Engine integration (Google, Bing, DuckDuckGo)
- Image search to clipboard
- Local capture History with SQLite storage
- Fully accessible 5-step Onboarding Wizard
- Comprehensive Settings window
- Dark/Light Theme Support
- Toast notification system
